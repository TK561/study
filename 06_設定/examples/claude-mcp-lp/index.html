セットアップ手順

5分で開発環境を構築

1. devcontainer環境の準備

VS Codeでdevcontainerを起動

code . # プロジェクトを開く
# Ctrl+Shift+P → "Dev Containers: Reopen in Container"

2. ホストマシンでMCPサーバー起動

PlaywrightとFigmaのMCPサーバーを起動

./scripts/start-mcp-servers.sh

3. Claude CodeでMCP接続

devcontainer内でMCPサーバーを登録

./scripts/setup-claude-mcp.sh

4. 開発開始

Claude Codeを起動して開発開始

claude
