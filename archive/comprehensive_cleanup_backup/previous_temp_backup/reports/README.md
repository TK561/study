# レポートディレクトリ

## フォルダ構成

### cleanup/
作業終了時の自動整理レポート
- `cleanup_report_YYYYMMDD_HHMM.md` - 日時別整理結果

### session/
セッション関連レポート（今後追加予定）
- 作業ログ
- パフォーマンス統計

## 自動整理について

作業終了時（「今日は終わり」等）に自動的に生成されます：
- ファイル整理結果
- 一時ファイル削除数
- ディスク使用量
- 整理前後の状況

## 古いレポートの管理

- 30日以上古いレポートは自動削除されます
- 重要なレポートは手動でバックアップしてください
